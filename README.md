# New port
> Get a free port

## Installation
```bash
npm install --save new-port
```

## Usage
```javascript
const newport = require('new-port');

newport.then(port => {
  // do something with port
});
```

> Working example in [test.js](https://github.com/tobihrbr/new-port/blob/master/test.js)
